## ðŸŽ® How to Play

1.  **Type Romaji**: Type the reading of a character (e.g., `ka`, `shi`, `tsu`) to deal a card.
2.  **Build a Hand**: Cards will pile up in the center. Try to type words or sentences!
3.  **Mistakes?**:
    - press **`Backspace`** to undo the last card.
    - press **`Space`** to clear the table and start over.
